# FUNIBER > Investigador > Abrir Proyectos > Desarrollo

## Implementación

El Investigador consulta únicamente proyectos en los que participa, incluidos los archivados para conservar el acceso al histórico y a sus documentos. El filtro se aplica en servidor usando la identidad de la sesión.
