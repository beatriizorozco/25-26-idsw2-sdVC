# Casos de uso - Análisis MVC

Esta carpeta contiene el análisis de colaboración de los casos de uso de la Plataforma Interna de Investigación de FUNIBER.

## Organización

- [Coordinador](coordinador/README.md): análisis de los casos de uso con permisos globales de coordinación.
- [Investigador](investigador/README.md): análisis de los casos de uso asociados a información propia y consultas permitidas.

## Convenciones

Cada caso de uso contiene:

- `README.md`: análisis MVC del caso de uso.
- `colaboracion.puml`: diagrama de colaboración en PlantUML.

Los análisis mantienen trazabilidad con `RUP/00-casos-uso/02-detalle` y preparan la posterior disciplina de diseño.
